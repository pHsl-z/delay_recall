# 托管班管理系统

这是一个完整的托管班管理Web应用，采用React 18 + Supabase架构。

## 快速开始

1. 点击 [这里](./index.html) 访问应用
2. 使用测试账号登录：13800000001/admin123

## 功能特色

- ✅ 完整的学生管理系统
- ✅ 收支管理和财务管理
- ✅ 考勤管理和成绩管理
- ✅ 作业管理和学习计划
- ✅ 通知管理和数据导出
- ✅ 现代化的UI设计

## 技术栈

- **前端框架**: React 18
- **样式系统**: Tailwind CSS + Glassmorphism
- **后端服务**: Supabase (PostgreSQL + Auth + Real-time)
- **图表库**: Chart.js
- **表格导出**: SheetJS (xlsx)
