## 数据库更新
为确保一键续费功能正常运行，需要确保数据库表包含以下字段：

```sql
-- students 表需要包含以下字段
ALTER TABLE students ADD COLUMN IF NOT EXISTS student_type VARCHAR(50) DEFAULT '月托';
ALTER TABLE students ADD COLUMN IF NOT EXISTS end_date DATE;
ALTER TABLE students ADD COLUMN IF NOT EXISTS status VARCHAR(20) DEFAULT '正常';

-- transactions 表需要包含以下字段
ALTER TABLE transactions ADD COLUMN IF NOT EXISTS notes TEXT;
ALTER TABLE transactions ADD COLUMN IF NOT EXISTS payment_method VARCHAR(50);
```

## 使用指南

### 一键续费操作步骤
1. 进入"学生管理"页面
2. 找到需要续费的学生
3. 点击学生行末的"续费"按钮
4. 确认续费信息（托管方式、续费周期、金额）
5. 点击"确认续费"完成操作

### 批量续费操作
1. 在学生列表中选择多个学生（复选框）
2. 点击顶部"批量续费"按钮
3. 设置续费参数
4. 确认批量续费信息
5. 执行批量续费操作

### 续费提醒设置
- 系统会自动标识即将到期的学生
- 支持自定义提醒提前天数
- 可在设置中配置提醒规则

## 注意事项
- 续费前请确认学生当前托管状态
- 批量续费操作不可撤销，请谨慎操作
- 续费记录会自动同步到财务系统
- 建议在续费后及时更新学生档案
